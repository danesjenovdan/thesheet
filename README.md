# danesjenovdan/thesheet

## Installation
 [Composer](https://getcomposer.org/).

run 
```
composer install
```

- get json certificate from console.google.com, apps, keys,...

- copy php/sample.config.php to php/config.php

- edit php/config.php
- change __LOCALURL
- change db access

run 
```
composer build
```

run 
```
composer demo
```
